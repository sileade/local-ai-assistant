# План обучения AI-моделей на инфраструктуре Scoliologic

## 1. Текущее состояние инфраструктуры
Кластер состоит из трех узлов с суммарной VRAM 100 ГБ:
- **ai-server (10.0.0.229)**: 68 ГБ VRAM (RTX A6000 + RTX 4000 SFF). Используется для тяжелых моделей (`llama3.1:70b`, `qwen2.5-coder:32b`).
- **ai002 (172.27.5.114)**: 32 ГБ VRAM (2x RTX 5070 Ti). Используется для быстрых и специализированных моделей (`medcomm-ai-v1`, `qwen2.5:14b`).
- **ai012 (172.27.5.150)**: Дополнительный узел для быстрых моделей.

## 2. Подготовка данных (Выполнено)
Мы успешно конвертировали экспорт базы знаний `scoliologic.ru` (65 Markdown-документов) в обучающий датасет.
- Извлечено **896 пар вопрос-ответ**.
- Данные добавлены в `golden_dataset.jsonl`.
- Тематика: Инфраструктура 1С, DevOps, Сети (VPN, Traefik), Проектный офис, Управляющий совет.

## 3. План обучения по ролям (Experts)

### 3.1. SysAdmin / DevOps Expert (`sclg-sysadmin`)
- **Базовая модель**: `gemma3:27b` (или `qwen2.5:14b` для скорости)
- **Целевой узел**: `ai-server` (порт 11437)
- **Данные**: 896 новых записей из базы знаний + существующие примеры tool calling.
- **Цель**: Модель должна знать всю архитектуру Scoliologic (IP-адреса, порты, VPN, Proxmox, MinIO) и уметь применять эти знания при анализе логов и метрик.

### 3.2. General Assistant (`sclg-general`)
- **Базовая модель**: `qwen2.5:14b`
- **Целевой узел**: `ai002` (порт 11434)
- **Данные**: Общие документы (Проектный офис, Управляющий совет).
- **Цель**: Быстрые ответы на общие вопросы по процессам компании на русском языке.

## 4. Инструкция по запуску обучения

Все скрипты и данные уже находятся в репозитории `sclg-ai-repo`. Для запуска обучения на вашей инфраструктуре выполните следующие шаги на Mac Mini:

1. **Обновите репозиторий**:
   ```bash
   cd ~/sclg-ai-repo
   git pull origin main
   ```

2. **Запустите полный цикл обучения для SysAdmin**:
   ```bash
   python3 training/train.py cycle --expert sysadmin --remote ai-server
   ```
   *Этот скрипт сгенерирует Modelfile с новыми знаниями Scoliologic, создаст модель и отправит её на `ai-server`.*

3. **Запустите обучение для General**:
   ```bash
   python3 training/train.py cycle --expert general --remote ai002
   ```

4. **Проверьте результаты тестирования**:
   Скрипт автоматически запустит тесты из `test_scenarios.json` для проверки форматов ответа, подавления `<think>` тегов и знания инфраструктуры.
